<div class="ps-block ps-callout">
  <div class="ps-callout-title">
    <i class="bi bi-lightbulb-fill"></i>
    Main Concept
  </div>
    The effect size is the <span class="ps-highlight-term">strength or magnitude</span> of difference between the groups, used to determine if the a statistically significant result is meaningful.
</div>

<div class="ps-divider"></div>

<div class="ps-plain">
  <div class="ps-plain-title">
    <i class="bi bi-translate"></i>
    In plain English
  </div>
  Statistical significance tells you a difference is unlikely to be due to chance, assuming the null hypothesis is true. Effect size tells you whether that difference is big enough to matter.
</div>

## Ways to calculate effect size

<div class="ps-myths">
  <div class="ps-myth">
    <div class="ps-myth-icon"><i class="bi bi-check-circle-fill"></i></div>
    <div class="ps-myth-body">
      <div class="ps-myth-title">Cohen's d</div>
      <div class="ps-myth-text">
        For comparing two groups (e.g. t-test)
        <ul class="ps-list">
            <li><span class="ps-dot"></span><span>0.2 to 0.3 = small effect</span></li>
            <li><span class="ps-dot"></span><span>0.5 = medium effect</span></li>
            <li><span class="ps-dot"></span><span>0.8 to biggter = large effect</span></li>
        </ul>
      </div>
    </div>
  </div>

  <div class="ps-myth">
    <div class="ps-myth-icon"><i class="bi bi-check-circle-fill"></i></div>
    <div class="ps-myth-body">
      <div class="ps-myth-title">Partial eta<sup>2</sup> (η<sub><var>p</var></sub><sup>2</sup>)</div>
      <div class="ps-myth-text">
        This is a measure of variance and can be used to calculate the proportion of variance in the dependent variable that is explained by the independent variable. It can also be used to represent the effect size.
        <ul class="ps-list">
            <li><span class="ps-dot"></span><span>η<sub><var>p</var></sub><sup>2</sup> = 0.01 is a small association.</span></li>
            <li><span class="ps-dot"></span><span>η<sub><var>p</var></sub><sup>2</sup> = 0.06 is a medium association.</span></li>
            <li><span class="ps-dot"></span><span>η<sub><var>p</var></sub><sup>2</sup> = 0.14 or larger is a large association.</span></li>
        </ul>
      </div>
    </div>
  </div>
