    We cannot usually measure an entire population, so we use a sample to estimate the true population parameter which is the real value that describes everyone. Because different samples would give slightly different results, we use confidence intervals to show the range of plausible values for that unknown parameter.

<div class="ps-block ps-def">
  <div class="ps-term">Confidence Interval</div>
  <div>
    The range of values that is likely to contain the value of an unknown population parameter. 
  </div>
</div>

<div class="ps-block ps-def">
  <div class="ps-term">Confidence Level</div>
  <div>
  <ul class="ps-list">
        <li><span class="ps-dot"></span><span>The percentage of times you expect to reproduce an estimate between the upper and lower bounds of the confidence interval.</span></li>
        <li><span class="ps-dot"></span><span>For example, if you have a confidence interval with a 95% confidence level, that means you are confident that 95 out of 100 times the estimate will fall between the extremes defined by the confidence interval.</span></li>
    </ul>
  </div>

<div class="ps-tile-row">
  <div class="ps-tile">
    <h4>What is a confidence interval?</h4>
    A range of values calculated from sample data that is likely to contain the true population parameter.
  </div>
  <div class="ps-tile">
    <h4>When should you use it?</h4>
    When you want to estimate the size of an effect and show the uncertainty around that estimate
  </div>
  <div class="ps-tile">
    <h4>How to interpret it</h4>
    The interval shows the range of values that are compatible with your data. Narrow intervals indicate greater precision; wide intervals indicate more uncertainty. If the interval excludes the null value, the result is statistically significant.
  </div>
